# AGENTS.md

## 语言偏好

Always respond in Simplified Chinese (简体中文)。代码、命令、文件名保持英文，描述和解释全部用中文。

## 行为准则

本项目同时维护 [CLAUDE.md](CLAUDE.md)，包含四条核心行为准则，Codex 亦应遵循：

1. **编码前先思考** — 不确定就提问，列出选项让人选，禁止瞎猜
2. **简约至上** — 只写最少必要代码，不过度设计
3. **外科手术式编辑** — 只改被要求改的部分，不顺手"优化"无关代码
4. **目标驱动执行** — 模糊指令 → 可验证验收标准 → 循环直到通过

完整行为准则见 [CLAUDE.md](CLAUDE.md)。

## 项目背景

这是一个**中文学术论文编辑审校**工作区，主要处理护理学研究论文（.docx 格式）。工作内容包括：

- 论文参考文献著录格式审校
- 学术论文内容编校审读
- PDF 审校报告生成
- Word 文档追踪修订与批注

## 工作流约定

### 文件命名规范
- 原始稿件：`编号-日期-编号-作者【编辑】.docx`
- 修订稿：`编号-日期-编号-作者【编辑】-修订稿.docx` 或加 `完` 结尾表示定稿
- 审校报告：放置于 `审查报告/` 子目录

### 常用工具
- **文档处理**：python-docx、pandoc、LibreOffice（.docx ↔ .md 互转）
- **参考文献审校**：项目已安装 `ref-format-checker` Claude Code 插件
- **PDF 处理**：pypdf / PyPDF2

### 编辑守则
- 所有对稿件的修改必须通过 Word 追踪修订（Track Changes）进行
- 批注应标注具体位置和修改理由
- 参考文献格式遵循 GB/T 7714-2015 标准
- 保持原文的学术风格和专业术语一致性
