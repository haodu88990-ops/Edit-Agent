# 参考文献格式检查器

## 严重性分级（审读的第一原则）

**参考文献错误分三个等级，严重性依次递减。审读时必须按此优先级逐级排查——低级别问题不可挤压高级别问题的审读时间。**

| 级别 | 名称 | 问题类型 | 后果 | 示例 |
|------|------|---------|------|------|
| **🔴 一级** | **张冠李戴** | DOI 指向的论文与稿件所述完全不符；文献主题与论文主题无关 | **学术诚信受损**：引用的文献根本不存在或完全无关，审稿人/编辑可能直接拒稿 | 心脏起搏器论文出现在压力性损伤共识中；DOI 解析后作者/标题/刊名全部对不上 |
| **🟡 二级** | **数据错漏** | 作者名拼写错误、卷期页码复制错误、DOI 断行缺失、中英条目数据不一致、英文标题为自译而非官方标题、引用序号缺失/重复 | **引用不可追溯**：读者无法定位到原文，或定位到错误的文章；国际数据库检索无法命中 | [9] 缺序号；[10] 英文卷期页码复制自 [9]；"stress injury" 应为 "pressure injury"；DOI 独立成段无法解析 |
| **🟢 三级** | **格式不规范** | 标点符号错误、中英文标点混用、空格缺失/多余、文献类型标识缺失、DOI 大小写 | **影响投稿通过**：编辑/排版阶段可能退回修改，但不影响学术实质 | 逗号后缺空格；中文全角冒号混入英文条目；`doi:` 应为 `DOI:` |

**⚡ 核心原则：宁可漏掉十个格式问题，不可放过一个张冠李戴。**

格式正确不代表数据正确。DOI 是校验参考文献真实性的唯一锚点——每一条含 DOI 的文献，必须验证 DOI、作者、标题、刊名四者的一致性。正则表达式能发现标点错误，但永远无法发现"这篇论文根本不是作者引用那篇"。

---

## 三阶段执行流程（必须严格按此顺序，不可调换）

```
第一阶段 🔴 数据准确性 — 查"张冠李戴"（学术诚信，不可跳过）
  ├── DOI 四要素核验（DOI、作者、标题、刊名交叉验证）     ← 审读的核心价值
  ├── 作者姓名逐条比对（DOI 联网核验真实作者名）
  ├── 中英条目交叉校验（中文条目数据 vs 英文翻译行）      ← 常见卷期页码复制错误
  └── 引用相关性分析 + 主题偏离标记                       ← 识别"风马牛不相及"文献

第二阶段 🟡 引用一致性 — 查"数据错漏"（引用可追溯性）
  ├── 正文引用数量 vs 文献列表条数
  ├── 漏引 / 多引检测
  ├── 重复条目检测
  └── 引用序号连续性 + 序号缺失

第三阶段 🟢 格式规范性 — 查"标点空格"（投稿合规）
  ├── 标点符号（正文中文半角 vs 参考文献英文半角）
  ├── 空格规范
  ├── 文献类型标识
  ├── DOI 大小写及空格
  └── 卷/期/页码格式
```

**⚠️ 不得在第一阶段完成前进入第二、三阶段。一级问题未排查完就去改空格，是本末倒置。**

## 第一阶段：数据准确性检查

### 1. DOI 四要素核验 🔴 一级风险

**这是审读的核心价值所在，也是审稿人与编辑最关注的问题。格式错了可以改，文献引用错了是学术诚信问题。**

每一条含 DOI 的英文参考文献，必须验证以下四个要素的一致性：

| 要素 | 验证方式 | 不匹配时的严重性 |
|------|----------|-----------------|
| DOI | 通过 `https://doi.org/` 解析，确认可访问 | 🔴 无法解析 → 文献不存在 |
| 第一作者 | 姓氏拼写是否与 DOI 解析结果一致 | 🔴 不一致 → 张冠李戴 |
| 标题 | 关键词/主题是否与 DOI 解析结果匹配 | 🔴 不匹配 → 完全不是同一篇 |
| 刊名 | 是否与 DOI 解析结果一致 | 🟡 刊名错但文章对 → 影响检索 |

**四要素中任一不匹配，都意味着读者无法通过稿件中提供的信息定位到正确文献。其中作者/标题不匹配是最严重的——说明稿件引用的根本不是这篇文献。**

#### 1.1 DOI 核查决策树

拿到一条含 DOI 的参考文献后，按以下逻辑判断：

```
DOI 能否解析？
  ├── 否 → 🔴 标记"DOI 无效"，需人工核实
  └── 是 → 解析结果的第一作者是否与稿件一致？
              ├── 否 → 🔴 模式 A/B/C：张冠李戴。稿件作者/标题/刊名至少一项错
              └── 是 → 解析结果的标题关键词是否与稿件一致？
                        ├── 否 → 🔴 模式 A：同一作者不同论文，DOI 张冠李戴
                        └── 是 → 解析结果的刊名是否与稿件一致？
                                  ├── 否 → 🟡 模式 B：刊名错，但文章对
                                  └── 是 → ✓ 四要素通过
```

#### 1.2 DOI 错配模式分类（按严重性排序）

**🔴 模式 A：DOI 正确，但稿件中的作者/标题是另一篇论文的（"张冠李戴"）**

这是最严重的一类错误。稿件声称引用了某篇论文，但 DOI 指向的完全是另一篇论文——读者追查 DOI 会发现引文不存在。

```
稿件原文：Tayyib N, Coyer F, Lewis P. Effectiveness of nurse-led versus multidisciplinary
          team models in pressure injury healing[J]. J Clin Nurs, 2023, 32(5-6):712-726.
          DOI: 10.1111/jocn.16421

DOI 解析：Artola M, Hernando A, Vidal O, et al. The role of specialist nurses in detecting
          spasticity and related symptoms in multiple sclerosis[J]. J Clin Nurs, 2023...

错误本质：稿件作者/标题对应的文献与 DOI 指向的文献是两篇完全不同的论文。
修复方式：
  1. 在 PubMed/CrossRef 中用稿件中的作者+标题关键词检索，找到真实 DOI
  2. 如该文献不存在 → 学术诚信风险，建议删除并替换为可验证文献
  3. 如该文献存在且 DOI 不同 → 替换为正确 DOI
```

**🔴 模式 B：DOI 正确，稿件中的刊名/卷期/DOI 全错**

```
稿件原文：Brouwers MC, Kho ME, Browman GP, et al. AGREEⅡ...[J].
          Canadian Medical Association Journal, 2010, 182(18):E839-E842.
          DOI: 10.1503/cmaj.090449

DOI 解析：实际发表在 J Clin Epidemiol, 2010, 63(12):1308-1311.
          DOI: 10.1016/j.jclinepi.2010.07.001

错误本质：AGREE II 在 CMAJ 曾有早期版本，但 2010 年正式版发表在 J Clin Epidemiol。
          稿件中的刊名、卷期、DOI 全部指向了错误来源。
修复方式：替换为 J Clin Epidemiol 的完整正确信息。
```

**🔴 模式 C：DOI 指向同一期刊的不同论文**

```
稿件原文：Smet S, Lahousse A, Van Damme N, et al. Comparison of instruments for
          measuring pressure ulcer healing[J]. Wound Repair Regen, 2021, 29(4):635-647.
          DOI: 10.1111/wrr.12921

DOI 解析：2021 WHS Abstract Session List Key[J]. Wound Repair Regen, 2021, 29(3):A1-A53.

错误本质：DOI 被复用到其他论文（常见于文献管理软件批量导入时错位）。
修复方式：在 PubMed 中按作者+标题检索，找到 Smet 论文的正确 DOI。
```

**🔴 模式 D：稿件中的文献与论文主题完全无关（"风马牛不相及"）**

```
稿件主题：压力性损伤护理
稿件中文献：Kazawa S, et al. Feasibility of a novel atrial mechanical sensing method
            for leadless atrioventricular synchronous pacing[J]. J Cardiovasc Electrophysiol,
            2024, 35(6):1115-1120. DOI: 10.1111/jce.16255

错误本质：心脏起搏器论文出现在压力性损伤护理共识中，二者无任何学科交集。
          极可能是文献管理软件（EndNote/Zotero）导入时编号错乱或错误替换。
修复方式：核实原文引用上下文，找到应该引用的正确文献并替换。如无法确认正确文献，删除。
```

**🟡 模式 E：无 DOI / DOI 无法解析**

无 DOI 的文献无法通过自动化方式验证，需标记为"未经核验"，建议作者补充 DOI 或替换为可验证文献。

#### 1.2 DOI 四要素核验的执行标准

	**核验范围（强制）：**
	- **所有参考文献必须 100% 逐条核验，不得遗漏任何一条。**
	- 含 DOI 的文献：通过多渠道核验策略（见 api_reference.md 第二节）确认 DOI、作者、标题、刊名四要素一致。
	- 无 DOI 的文献：通过渠道 2（学术数据库标题检索）+ 渠道 3（WebSearch 多搜索词并行）主动核验，不得直接标记为"未经核验"跳过。
	- **禁止以"覆盖率已达 X%"为由跳过任何一条参考文献。**
	- 核验顺序按以下优先级排列（不改变逐条核验要求，仅决定先后）：
	  - 第一作者看不出中文姓氏的华人团队论文
	  - 刊名使用全称但不确定是否为官方名称的
	  - 标题与论文主题关联度存疑的
	  - DOI 后缀格式异常的（如 `0012`、`07.0` 等）

**核验方法（详见 api_reference.md 第二节"参考文献核验自动化策略"）：**
```
渠道 1（有 DOI 优先）：doi.org CSL JSON API → CrossRef API / PubMed
渠道 2（不限，API 优先）：PubMed 标题检索 → OpenAlex → Semantic Scholar
渠道 3（兜底核心）：WebSearch 多搜索词并行（6 条搜索词，任一条命中即可）
  · "{DOI}" / site:xueshu.baidu.com / site:med.wanfangdata.com.cn /
    "{英文标题}" / "{作者} {刊名} {年份}" / "{完整中文标题}"
渠道 4：CNKI DOI 结构分析
渠道 5：万方/知网/维普人工核验（渠道 1-4 均失败时）

任一渠道核验通过即确认该条文献正确。
与稿件参考文献逐项比对，四要素中任一项不匹配 → 标记为问题条目。
```

**核验记录格式：**
```
【参考文献核验】
核验总数：X 条（全部参考文献必须 100% 逐条核验）
✓ 通过：A 条
✗ 不匹配：B 条
  - [N] 模式X：具体问题描述（DOI: xxx）
⚠ 无法确认：C 条（自动化渠道 1-4 均失败，需人工核验）
```

### 2. 作者姓名逐条核验

对英文参考文献的每位作者，**逐条、逐位核验**，不得因"看起来没问题"而跳过：

**核验范围：** 所有英文参考文献的全部作者，而非仅"可疑条目"。

**核验要点：**
- 华人作者名是否符合拼音规则（如 `Jing J W` 而非 `Wang JJ`）
- 名字像姓氏、姓氏像名字的颠倒情况（如 `Paige H. D` 实为 `Dean PH`）
- 作者名缩写中出现不常见格式（中间名缩写、连字符异常）
- 来自中国研究团队但第一作者看不出中文姓氏
- 同一作者在不同文献中拼写是否一致

**核验方式：** 逐条通过 DOI 联网获取真实作者列表，逐位比对。含 DOI 的文献一律核验，不得只核验"红旗信号"条目。

### 3. 中英条目交叉校验

中文期刊论文通常需要提供英文翻译行。**必须对照中文条目，逐项验证英文翻译行数据。**

**此项检查的重要性不亚于 DOI 核验**——英文翻译行是国际数据库（PubMed/Scopus/WoS）检索的唯一入口。翻译行的错误直接导致论文在国际数据库中无法被检索到，等同于文献引用失效。

#### 3.1 英文标题来源检测（自译 vs 官方标题）🔴🟡

中文期刊论文的英文标题有两种来源：
- **官方标题**：期刊官网/DOI 注册时登记的标准英文标题（术语准确、格式规范）
- **自译标题**：作者自行翻译的英文标题（术语不统一、表述生硬、刊名瞎翻）

**自译标题的五大红旗信号：**

| 红旗信号 | 示例（错误 → 正确） | 严重性 |
|---------|---------------------|--------|
| 术语自造，与学术惯例不符 | "stress injury" → "pressure injury" | 🔴 国际检索无法命中 |
| 刊名英文为自译而非官方名 | "Modern Chinese doctors" → "China Modern Doctor" | 🔴 刊名对不上的文献等于不存在 |
| 刊名英文缺词/多词 | "Journal of Nursing" → "Journal of Nursing Science" | 🟡 检索困难 |
| 标题语法生硬、逐字直译痕迹明显 | "Applied research on the intervention strategy of secondary healing..." | 🟡 非母语表达 |
| 英文标题直接缺失 | 中文条目有英文翻译行但无英文标题 | 🟡 需补充 |

**检测方法：**
```
1. 对每条含中文条目的英文翻译行，提取英文标题
2. 通过 WebSearch 搜索"{中文标题关键词} {英文期刊名}"，从期刊官网/PubMed/万方
   获取该论文的官方英文标题
3. 对比稿件英文标题与官方英文标题：
   - 完全一致 → ✅ 官方标题
   - 仅标点/大小写差异 → 🟡 可能是复制时的格式变异
   - 用词不同（术语、语序、刊名） → 🔴 自译标题，需替换为官方版本
   - 稿件无英文标题但期刊官网有 → 🟡 遗漏，需补充官方英文标题
```

**核验记录格式：**
```
【英文标题来源核验】
核验 X 条中文期刊英文翻译行
✅ 官方标题一致：A 条
🔴 自译标题（术语/刊名错误）：B 条
  - [N] 术语错误：stress injury → pressure injury
  - [N] 刊名自译：Modern Chinese doctors → China Modern Doctor
🟡 自译标题（语法生硬但可理解）：C 条
🟡 英文标题缺失：D 条（需补充）
```

#### 3.2 卷期页码交叉校验

**常见错误：英文翻译行的卷期页码复制自相邻文献**（如 [10] 英文写了 [9] 的数据）。必须逐条对照中文原文核验。

```
中文：[10] 周英凤, 顾莺, 胡雁, 等. JBI循证卫生保健中心关于不同类型研究的
       质量评价工具：干预性研究的质量评价[J]. 护士进修杂志, 2018, 33(1):24-26.

英文：Zhou YF, Gu Y, Hu Y, et al. Quality assessment tool...J Nurs Train, 2018, 33(8):701-703.
       ↑ 卷期页码 33(8):701-703 复制自 [9]，应为 33(1):24-26

检测方法：逐条提取中文条目的(年,卷,期,页)与英文翻译行比对，不一致的立即标记。
```

#### 3.3 英文刊名核验

中文期刊的官方英文名称是固定的，不可自行翻译。常见错误：

| 中文刊名 | 正确英文名 | 常见自译错误 |
|---------|-----------|-------------|
| 护理学杂志 | Journal of Nursing Science | Journal of Nursing（缺 Science） |
| 中华护理杂志 | Chinese Journal of Nursing / Chin J Nurs | China Journal of Nursing |
| 护理学报 | Journal of Nursing (China) / J Nurs | Journal of Nursing |
| 护理研究 | Chinese Nursing Research | Nursing Research |
| 中国护理管理 | Chinese Nursing Management | China Nursing Management |
| 护士进修杂志 | Journal of Nurses Training | Nurses Training Journal |
| 中西医结合护理 | Journal of Integrative Nursing / Chin J Integr Nurs | Journal of Clinical Nursing in Practice |
| 中国现代医生 | China Modern Doctor | Modern Chinese doctors |
| 中国卫生统计 | Chinese Journal of Health Statistics | China Health Statistics |
| 中国组织工程研究 | Chinese Journal of Tissue Engineering Research | China Tissue Engineering Research |

**检测方法：**
```
1. 识别英文翻译行中的刊名（[J]. 和 ,{年} 之间的部分）
2. 与上表比对，标记不一致的
3. 对上表中未收录的期刊，通过 WebSearch 搜索"{中文刊名} English name"
   获取官方英文名称
```

#### 3.4 英文标题中术语一致性检查

全文所有英文翻译行中的核心术语必须统一。在护理学稿件中，以下术语是高频错误点：

| 中文术语 | 正确英文 | 常见错误翻译 |
|---------|---------|-------------|
| 压力性损伤 | pressure injury | stress injury / pressure ulcer（过时术语） |
| 压疮 | pressure ulcer / pressure sore | bedsore / decubitus |
| 失能老人 | disabled elderly / elderly with disabilities | disabled old people |
| 互联网+护理服务 | Internet Plus nursing service | Internet and nursing / online nursing |
| 循证护理 | evidence-based nursing | evidence nursing |
| 系统评价 | systematic review | systematic evaluation |
| Meta 分析 | meta-analysis | Meta analysis |

**检测方法**：在英文翻译行中搜索错误术语列表，标记所有命中项。

### 4. 引用相关性分析

**检查每篇参考文献的标题/主题是否与论文主题相关：**
- 阅读文献标题，判断是否与论文关键词（如"压力性损伤""互联网+护理""失能老人"等）相关
- 对疑似不相关的文献，通过 DOI 解析确认其真实内容
- 标记所有"主题偏离"文献，建议核实或替换

**同时检查：**
- 文献时效性：近 5 年（2020 年后）文献占比是否 ≥ 60%
- 超过 10 年的文献需特别注明保留理由（经典文献、方法学奠基性文献、专著等可豁免）

## 第二阶段：引用一致性检查

### 检查步骤

1. 提取正文中所有引用标记（`[1]`、`[2-5]`、`[6,8]` 等）
2. 展开范围引用，去重，得到正文引用的全部序号
3. 提取参考文献列表中的所有条目序号
4. 对比：
   - 正文引用但参考文献缺失 → **漏引**
   - 参考文献有但正文未引用 → **多引**
   - 引用序号不连续 → 检查是否合理（如 [1][3] 缺 [2]）
   - 最大引用序号 > 参考文献总数 → **严重错误**

## 第三阶段：格式规范性检查

### 1. 标点符号规则

| 位置 | 规则 | 示例 |
|------|------|------|
| 正文中 | 中文半角标点 | `，。：；` |
| 参考文献中 | **英文半角**标点 | `, . : ;` |
| 作者分隔 | 英文逗号 `,` + 空格 | `Zhang Y, Liu X, Wang L` |
| 刊名后 | 英文逗号 `,` + 空格 + 年 | `J Nurs, 2024,` |
| 年后 | 英文逗号 `,` + 空格 + 卷 | `2024, 59(3):` |
| 卷(期)后 | 英文冒号 `:` + 页码 | `59(3):345-349` |
| 页码后 | 英文句号 `.` | `345-349.` |
| DOI 前缀 | 大写 `DOI:` + 空格 | `DOI: 10.xxx` |

### 2. 高频错误检测模式

| 错误 | 正则/检测方式 | 说明 |
|------|-------------|------|
| 中文冒号混入英文参考文献 | `：` 出现在参考文献段 | 应为 `:` |
| 逗号后缺空格 | `,\w` 模式（逗号直接跟字母） | 应为 `, ` |
| DOI 前缺空格 | `\.DOI:` | 应为 `. DOI:` |
| DOI 小写 | `doi:` | 应为 `DOI:` |
| 年份后句号非逗号 | `\d{4}\.\s*\d+\(` | 应为 `, ` |
| 刊名与年份间缺空格 | `[J]\. \w+, \d{4}` 以外的模式 | 检查 |
| 单词连写 | `clinicalpractice`、`thetreatment` | 应分开 |
| "et al" 前缺空格 | `\w,et al` | 应为 `, et al` |
| 行末多余中文标点 | `，$` `。$` 出现在参考文献段 | 删除 |

### 3. 文献类型标识检查

参考文献必须包含类型标识：`[J]` `[M]` `[D]` `[R]` `[S]` `[EB/OL]` `[C]` `[P]`

若条目无标识，根据文献特征推断并建议补充。

### 4. 作者格式

- 中文：姓名全称，3 人以上用"等"
- 英文：姓全称 + 名首字母缩写，3 人以上用 "et al"
- 英文作者名首字母缩写后不加点（`Zhang Y`）或统一加点（`Zhang Y.`），全文保持一致

### 5. 卷期页码格式

- 卷(期)：如 `59(3)`
- 页码：`起始页-结束页`
- 缺卷号：`2015(11)` → 应查明卷号，补为 `2015, 30(11)`
